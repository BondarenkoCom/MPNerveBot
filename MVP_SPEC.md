# MVP Specification

## Product Scope

Public-safe showcase of a Telegram marketplace operations bot.

## Roles

- `Owner`
- `Operator`

## Commands

- `/start`
- `/digest`
- `/reviews`
- `/alerts`
- `/status`
- `/help`

## Core Flows

- onboarding
- digest delivery
- alert visibility
- review follow-up

## Provider Naming

The public repo uses generic labels:

- `Marketplace A`
- `Marketplace B`

## V1

- onboarding
- demo mode
- generic account persistence
- digest and alerts
- review workflow shell

## V2

- richer AI layer
- stronger scheduling
- private provider-specific adapters
